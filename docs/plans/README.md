# Plans

Claude Code plan files — auto-generated during planning sessions.

Содержание каталога:
- `spec-*.md` — планы реализации задач
- `audit-fixes-*.md` — планы аудита и исправлений
- Команды для выполнения доступны через slash commands:
  - `/spec` — безопасное планирование (избегает Plan Mode bug)
  - `/compact-edw` — компакция с сохранением EDW-критического состояния
