### 6. Data Integrity (Pandas)
- ALWAYS: `dtype=str, na_filter=False` when reading CSV from `data/current/`
- Convert NaT/empty strings to `None` BEFORE DB insertion
- Use `.strip()` or `None` pattern for string extractions
- Preserve original values in `_raw` columns on FK failures
- Use `data/_history/` for archival

### 7. SCD Type 2 Logic (Tag Synchronization)
- Every change in `project_core.tag` → logged in `project_core.tag_history`
- Status: `NEW`, `UPDATED`, `DELETED`
- Compare `row_hash` before INSERT/UPDATE (minimize writes)
- Hierarchy resolution (`tag_parent_id`) happens AFTER main sync (same flow)
- Migration: `sql/migrations/002_add_tag_history.sql`
